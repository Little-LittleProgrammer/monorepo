# 前言

## 版本
- 本目录主要是提供公共的hooks方法
- npm包名称 `@quantum-design/hooks`
- 当前版本: 1.0.7

:::danger 提示
重要重要重要
- `1.1.x` 版本进行破坏性升级，`vue3.2.x`及`vite3.2.5`用户请使用 `1.0.x`版本
:::
