# 切量

## 定义
切量代表切出一部分用户, 走专门的特殊配制, 里面融合了目前广告运营功能的所有部分, 所以比较复杂.
流程: 先框出一波人(分桶) -》 根据广告位配置具体策略 -》 框出的这部分人是否在配置的定向分组中 -》再基于前面的定向配置定向分组 -》广告位策略分组 -》 流量下发组分组 -》特殊字段 -》 其他策略


## 前端实现

### 公共组件
> 因所有公共业务组件已经原子化, 所以当需要用到哪一部分组件直接使用即可

2. 策略定向功能(2)

5. 其他策略功能(13)
